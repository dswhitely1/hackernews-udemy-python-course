## Install

### React

```sh
cd client && yarn install
```

### Python
```sh
cd server
pipenv install
pip install -r requirements.txt
```

## Usage

### React

```sh
cd client && yarn start
```

### Python

```sh
cd server
pipenv shell
flask run
```

<p>You will need to set two environment varialble</p>

```sh
FLASK_APP=server.py
FLASK_ENV=development
```
